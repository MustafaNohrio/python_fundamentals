# Python Fundamentals

This repository contains my Python learning exercises, practice programs, and mini-projects as I build the foundation required for Machine Learning and AI.

## Goals

* Learn Python fundamentals
* Build problem-solving skills
* Practice writing clean and readable code
* Prepare for NumPy, Pandas, and Machine Learning

## Repository Structure

```text
python_fundamentals/
```

* Practice exercises
